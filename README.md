# FinOps_CS_Unified_model
A unified customer touchpoint model aimed at trackin operational KPIs in customer support domain.

The objective of this model was to provide easy access and for analysing the performance of the customer support department for the european markets

The KPIs that managment reqired to track were the following:
- CSAT (Customer Satisfaction Score)
- Call Acceptance Rate
- Chat Acceptance Rate
- Email Acceptance Rate
- Case Resolution Rate
- Cost per Touchpoint

Raw data sources that were provided:
- Five9 (call data)
- Whatsapp (chat data)
- Salesforce (case data, email data)
- Merchants (customer data)
